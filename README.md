# Protocol Atlas

> Understanding the open web through protocols, source code, and experiments.

Protocol Atlas is not a cryptocurrency encyclopedia, nor does it provide investment advice.
It explores a more fundamental question:

**How does the open web maintain state, reach consensus, and defend against malicious behavior in the absence of a centralized trusted authority?**

## Methodology

Each topic follows the same chain of evidence:

> Problem → Protocol Rules → Source Code Implementation → Reproducible Experiments → Attack Boundaries → Cross-Protocol Comparisons

## Current Research Tracks

- [ ] Bitcoin: Double-spending, UTXO, Transaction Validation, Blocks, PoW, P2P, Incentive Mechanisms
- [ ] Ethereum: Account model, EVM, Gas, PoS, finality
- [ ] Cross-protocol: Design trade-offs for the same problem across different protocols

## Principles

- Prioritize citing original sources, specifications, and client source code.
- Clearly distinguish between “protocol consensus rules,” “node strategies,” and “personal interpretations.”
- Provide verifiable pathways for each conclusion whenever possible.
- Do not upload private keys, mnemonic phrases, API keys, or real financial information.
