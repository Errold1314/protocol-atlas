# Implementation and Terminology

## Current Stage

At this stage, I have only created a glossary; I have not yet completed the Bitcoin Core source code walkthrough.

- UTXO: Unspent Transaction Output
- mempool: A set of candidate transactions temporarily stored by a node
- validation: A node’s verification of transactions and blocks according to consensus rules
- chainwork: Cumulative work on the chain
